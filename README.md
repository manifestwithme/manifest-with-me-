# ✨ Manifest With Me — Free Deployment Guide

Your app is ready! Here's how to get it live in under 10 minutes for **$0**.

---

## 🚀 Step 1: Create a Free GitHub Account
Go to **github.com** and sign up (free).

---

## 📁 Step 2: Create a New Repository
1. Click the **+** icon → **New repository**
2. Name it: `manifest-with-me` (or anything you like)
3. Set it to **Public**
4. Click **Create repository**

---

## ⬆️ Step 3: Upload Your Files
1. In your new repo, click **Add file → Upload files**
2. Upload all 4 files from this folder:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `README.md`
3. Click **Commit changes**

---

## 🌐 Step 4: Enable GitHub Pages (Makes it live!)
1. Go to your repo → **Settings** tab
2. Scroll to **Pages** in the left sidebar
3. Under **Source**, select **main** branch and click **Save**
4. Wait ~60 seconds, then your app is live at:
   **`https://YOUR-USERNAME.github.io/manifest-with-me`**

---

## 📸 Step 5: Link from Instagram
1. Copy your live URL from GitHub Pages
2. Go to your Instagram profile → **Edit Profile**
3. Paste the URL in the **Website** field
4. Save — now your followers can tap straight to the app!

---

## 💰 Step 6: Set Up Payments (When Ready)
For the PRO tier, use one of these **free platforms** (they only charge when you earn):
- **Ko-fi** (ko-fi.com) — one-time payments, 0% fee
- **Gumroad** (gumroad.com) — simple subscriptions, small % fee
- **Patreon** (patreon.com) — monthly memberships

Then update the `handleUpgrade()` function in `index.html` to link to your payment page.

---

## 🎨 Customizing Your App
Everything is in `index.html`. Easy things to change:
- **Colors**: Search for `#8b5cf6` (purple) or `#ec4899` (pink) to change the theme
- **Affirmations**: Find the `AFFIRMATIONS` object near the top of the `<script>` tag
- **Your Instagram link**: Search for `manifesting_with_me_today`
- **App name**: Search for `Manifest With Me`

---

## 📱 Users Can Install It Like a Real App
**iPhone:** Open in Safari → Share button → "Add to Home Screen"
**Android:** Open in Chrome → Three dots menu → "Add to Home Screen" or "Install App"
